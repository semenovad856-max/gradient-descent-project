# Описание данных

## 1. Синтетические данные для регрессии
- **Файл:** `regression_data.csv`
- **Размер:** 500 примеров
- **Признаки:** 1 числовой признак (feature)
- **Целевая переменная:** target (непрерывная)
- **Генерация:** с помощью `make_regression` из scikit-learn
- **Параметры:** линейная зависимость с добавлением шума (noise=15)
- **Назначение:** тестирование методов градиентного спуска для задачи линейной регрессии с MSE

## 2. Синтетические данные для классификации
- **Файл:** `classification_data.csv`
- **Размер:** 500 примеров
- **Признаки:** 2 числовых признака (feature1, feature2)
- **Целевая переменная:** target (бинарная: 0 или 1)
- **Генерация:** с помощью `make_classification` из scikit-learn
- **Параметры:** линейно разделимые классы с небольшим перекрытием
- **Назначение:** тестирование методов градиентного спуска для задачи логистической классификации с LogLoss

## Примечания
- Данные сгенерированы с фиксированным `random_state=42` для воспроизводимости
- Используются только для экспериментальной части проекта
- Реальные данные не требуются, так как цель — сравнение алгоритмов оптимизации
