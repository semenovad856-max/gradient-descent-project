# Кастомная реализация и компаративный анализ методов градиентного спуска

**Экзаменационный проект**

## Описание проекта
Самостоятельная (кастомная) реализация алгоритмов градиентного спуска и их применение к задачам линейной и логистической регрессии. Сравнительный анализ различных методов оптимизации.

## Архитектура проекта
gradient-descent-project/
├── src/ # Исходный код кастомных реализаций
├── experiments/ # Скрипты для проведения экспериментов
├── data/ # Наборы данных (синтетические и реальные)
├── docs/ # Документация, схемы, блок-схемы
├── results/ # Графики, таблицы, результаты сравнения
├── README.md # Это файл
├── requirements.txt # Зависимости Python
├── .gitignore # Игнорируемые файлы
└── LICENSE # Лицензия MIT


## Технологии
- **Python 3.10+**
- **NumPy** — математические вычисления
- **Matplotlib** — визуализация результатов
- **scikit-learn** — только для загрузки данных и сравнения
- **Кастомная реализация** — без использования готовых оптимизаторов

## Методы для реализации
1. **Пакетный градиентный спуск (Batch GD)** - обновление по всему набору данных
2. **Стохастический градиентный спуск (SGD)** - обновление по одному примеру
3. **Мини-пакетный градиентный спуск (Mini-batch GD)** - обновление по небольшой выборке
4. **SGD с моментом (Momentum)** - SGD с инерцией для ускорения сходимости

## Задачи
- **Регрессия:** Линейная регрессия с MSE
- **Классификация:** Логистическая регрессия с LogLoss
